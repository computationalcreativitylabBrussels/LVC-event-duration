# How Long Does a Quick Kiss Take? Studying Event Duration of Light Verb Constructions Using Explicit Word Embeddings

# Requirements
All the required Python libraries are listed in `code/env.yml`.

# Usage Example
To run the code in `/code/explore_vectorspace/semantic_projection.py` execute the command below in your terminal (after activating your own Python virtual environment with all the required dependencies installed).

Replace `PATH-TO` by the path to your local copy of this code repository.

```python explore_vectorspace/semantic_projection.py --vectorspace=PATH-TO/LVC-event-duration/code/io/ppmi.txt --basis-dimension=2000 --lvcs=PATH-TO/LVC-event-duration/code/io/LVC-FVC-pairs.csv --projection-words=PATH-TO/LVC-event-duration/code/io/duration_liu_chersoni.csv --output=PATH-TO/LVC-event-duration/code/io/projection_plain_verbs```
